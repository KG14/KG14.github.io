Past Summer Camps

Android App Development

An introduction to app development with Android.

Skill level: INTERMEDIATE

Monday, June 21, to Friday, June 25

2:00 PM to 4:00 PM every day (10 hours)

 This FREE camp will introduce students to the basics of app development with Android. Students will learn to create Android apps using Android Studio, Java, and XML, along with a number of Android front-end and back-end components. They will end the camp with a final project: building a note-taking app from scratch, complete with an attractive UI and functional back-end that allows notes to be stored in a database. Basic knowledge of Java is highly recommended.

NOTE: This camp is now full. Enrollment is closed.

More info

NOTE: Basic knowledge of the Java language is highly recommended for this camp.

Students will also need access to a computer.

The Android Studio IDE can be installed from here, but we will cover the installation in the camp as well.

Intro to Python

An introduction to programming in Python.

Skill level: BEGINNER

Monday, July 12, to Friday, July 16

10:00 AM to 12:00 PM every day (10 hours)

 This FREE camp will introduce students to the basics of programming in Python. Students will learn foundational concepts such as program structure, data types, variables, basic operations, logic with conditionals, loops, data structures like arrays, functions, and essential library functions. They will have a chance to use these new skills in a number of mini-projects and will end the camp with a final project: building the game TicTacToe with Pygame graphics.

NOTE: This camp is now full. Enrollment is closed.

More info

NOTE: No prior knowledge of Python is required; the camp will introduce students to the language.

Students will also need access to a computer.

Python can be downloaded from here, but we will cover the installation in the camp as well.
